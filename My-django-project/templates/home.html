API Endpoints

GET /health/
Health check endpoint
POST /api/stk-push/
Initiate STK push payment
Request Body: 
{
  "phone": "254700686463",
  "amount": 10,
  "reference": "Order #123"
}

M-Pesa callback handler

Django admin interface

Quick Start

1. Health Check
curl https://my-django-project-1-0k73.onrender.com/health/
2. Initiate Payment
curl -X POST https://my-django-project-1-0k73.onrender.com/api/stk-push/ \
  -H "Content-Type: application/json" \
  -d '{"phone":"254700686463","amount":10,"reference":"Test"}'
3. Access Admin Dashboard

Visit /admin/ to manage transactions, view payment history, and configure settings.
